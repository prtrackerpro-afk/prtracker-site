## O que é um PR (Personal Record)

PR (Personal Record) é o melhor desempenho individual já alcançado por um atleta em um determinado movimento ou exercício.

Exemplos:
- Maior carga levantada (ex: deadlift 180 kg)
- Melhor tempo em um treino específico
- Execução mais eficiente de um movimento

Na PR Tracker, o foco principal é **PRs de força** (levantamento de peso).

---

## O que é a PR Tracker

A PR Tracker é uma marca de miniaturas premium que transformam conquistas no treino em troféus físicos. O produto não é um acessório funcional de academia — é uma peça decorativa e emocional, criada para representar um PR de forma tangível e permanente.

A ideia central: a pessoa olha para a peça e lembra de uma conquista específica no treino. A academia deixa de ser apenas um lugar para melhorar o bem-estar e passa a ser um lugar para compartilhar conquistas.

A PR Tracker vem como uma grande inovação para o mercado brasileiro, funcionando também como marketing afetivo para boxes de CrossFit.

**Não é um souvenir. Não é um brinquedo. É um troféu.**

1. Envio do produto ao creator.

3. Audiência reconhece a conquista que já viveu.
